# Real Estate Agency Portal Starter Files (Spring 2026)

This starter project is a PHP + MySQL scaffold for the **Real Estate Agency Portal** project.

## Included
- Basic folder structure
- Database connection class
- Authentication scaffold
- Property listing scaffold
- Inquiry form scaffold
- Dashboard scaffold
- SQL starter script

## Student Tasks
Students should:
1. Create the MySQL database and tables.
2. Complete the TODOs in the PHP files.
3. Implement secure login and registration.
4. Add role-based behavior for agents, buyers, and renters.
5. Connect forms to the database.
6. Test the full application locally.

## Suggested Local Setup
- XAMPP / MAMP / WAMP / Laragon
- PHP 8+
- MySQL / MariaDB

## Default Structure
- `config/`
- `classes/`
- `includes/`
- `sql/`
- root PHP pages
